# Sample Data

This folder contains sample datasets for testing the Multi-Agent AI System.

## Files

| File | Description | Use Case |
|------|-------------|----------|
| `sales_data.csv` | Sales transactions with products, revenue, and regions | Data Intelligence Agent |
| `customer_data.csv` | Customer profiles with demographics | Data Intelligence Agent |
| `financial_data.xlsx` | Financial records in Excel format | Data Intelligence Agent |
| `sample_research_paper.pdf` | Research paper on computer vision | Research Assistant Agent |
| `sample_research_paper.txt` | Text version of the research paper | Research Assistant Agent |

## Usage

1. Upload any of these files through the web interface
2. Ask natural language questions about the data
3. The system will automatically route your query to the appropriate agent

## Example Queries

**For Sales Data:**
- "What is the total revenue?"
- "Show top 5 products by sales"
- "Plot revenue by region"

**For Research Paper:**
- "Summarize this document"
- "What are the key findings?"
- "Extract main topics"
